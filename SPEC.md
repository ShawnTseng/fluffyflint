
# Fluffy Flint 個人網站規格書（依 index.html 實作內容修訂）

## 1. 導覽列（Nav Bar）
- 固定於頁面頂部，半透明白色背景，圓角，帶陰影
- 左側：Logo「FF」字樣
- 右側：五個導覽連結（HOME、WORKS、BLOG、ABOUT、CONTACTS）
    - 當前頁面高亮（黃色底線）
- 內容左右分佈，寬度 80vw（最大 1200px），與主內容對齊
- 桌面版為主，行動裝置下寬度自動縮放，尚未有漢堡選單

## 2. 首頁 Banner
- 背景：彩色木質玩具高質感照片，滿版（Unsplash 圖片）
- 置中大標題：「Hello! This Is Fluffy Flint.」
- 副標題：簡短介紹文字，字體較小
- 主要按鈕：「More」，圓角、帶邊框，置中於標題下方，點擊滑動至作品區
- 向下箭頭提示可繼續捲動，點擊同樣滑動至作品區

## 3. 作品區（Projects Section）
- 位置：Banner 下方
- 標題：「Projects」置中、粗體
- 副標題：簡短說明，置中
- 作品卡片橫向排列（桌面每排四欄，RWD 單欄）
    - 編號（如 01, 02, ...）
    - 作品名稱
    - 作品圖片（全覆蓋卡片背景，半透明遮罩）
    - 作品簡介
    - 標籤（如 #Sustainable #Wood #Playful）
    - 「Learn More」按鈕，帶箭頭（目前無跳轉功能）
    - 滑鼠 hover 效果（亮度變化、陰影）
- 下方有「More Projects」按鈕（目前無跳轉功能）

## 4. 其他頁面（Blog, About, Contacts）
- 僅有導覽列連結，尚未有實際頁面內容（僅錨點）

## 5. 字體與配色
- 字體：Noto Sans（Google Fonts）、Arial、Helvetica、無襯線
- 主要色：白、黑、灰，搭配彩色玩具圖片
- 強調色：導覽列選中項目下方黃色底線（#ffe066）

## 6. 響應式設計
- 桌面版為主，兼容平板與手機（RWD）
- 行動裝置導覽列目前僅縮排，尚未有漢堡選單

## 7. 互動
- 導覽列點擊跳轉對應區塊（錨點）
- 「More」與下箭頭按鈕滑動至作品區
- 「Learn More」按鈕目前無跳轉功能

## 8. 頁尾（Footer）
- 置中顯示設計者名稱（HE TING YI）
- 下方有四個導覽連結（HOME、WORKS、ABOUT、CONTACT）
- 社群連結（Behance、Upwork、LinkedIn、Instagram、Mail，僅為佔位）
