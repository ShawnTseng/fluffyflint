# Fluffy Flint 個人網站規格書

## 1. 導覽列（Nav Bar）
- 固定於頁面頂部，半透明白色背景，圓角，帶陰影
- 左側：Logo 或「FF」字樣
- 右側：五個導覽連結（HOME、WORKS、BLOG、ABOUT、CONTACTS）
    - 當前頁面高亮（如底線或顏色）
- 內容左右分佈，寬度與主內容對齊

## 2. 首頁 Banner
- 背景：彩色木質玩具高質感照片，滿版
- 置中大標題：「Hello! This Is Fluffy Flint.」
- 副標題：簡短介紹文字，字體較小
- 主要按鈕：「More」，圓角、帶邊框，置中於標題下方
- 向下箭頭提示可繼續捲動

## 3. 作品區（Projects Section）
- 位置：Banner 下方
- 標題：「Projects」置中、粗體
- 副標題：簡短說明，置中
- 作品卡片橫向排列（每排四欄）
    - 編號（如 01, 02, ...）
    - 作品名稱
    - 作品圖片（全覆蓋卡片背景，半透明遮罩）
    - 作品簡介
    - 標籤（如 #Sustainable #Wood #Playful）
    - 「Learn More」按鈕，帶箭頭
    - 滑鼠 hover 效果（如亮度變化或陰影）

## 4. 其他頁面（Blog, About, Contacts）
- 目前為簡單佔位頁面，後續可擴充

## 5. 字體與配色
- 字體：無襯線（如 Noto Sans, Arial, Helvetica, Google Fonts 類似字體）
- 主要色：白、黑、灰，搭配彩色玩具圖片
- 強調色：導覽列選中項目下方黃色底線

## 6. 響應式設計
- 桌面版為主，兼容平板與手機
- 行動裝置導覽列可收合為漢堡選單

## 7. 互動
- 導覽列點擊跳轉對應區塊或頁面
- 「More」按鈕滑動至作品區
- 「Learn More」按鈕進入作品詳細頁（可先為佔位）
